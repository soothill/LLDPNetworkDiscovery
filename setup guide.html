üìã Overview

The LLDP Network Topology Generator is a powerful Python script that connects to network devices, retrieves LLDP (Link Layer Discovery Protocol) information, and generates comprehensive reports and visual network topology diagrams.

üîê Multiple Auth Methods

Password, SSH keys, or hybrid authentication with device-specific credentials

üåê Multi-Vendor Support

Cisco, MikroTik, HP/Aruba, Linux/Proxmox, and Arista devices

üìä Multiple Reports

CSV, HTML tables, static diagrams, and interactive visualizations

üé® Visual Topology

Automatic network diagram generation with device type detection

üîç Troubleshooting

Connection Timeout

Problem: Script times out when connecting to devices
Solutions:
  • Verify device IP addresses are correct and reachable
  • Check firewall rules allow SSH (port 22)
  • Verify SSH is enabled on devices
  • Test manual SSH connection: ssh username@device-ip

Authentication Failed

Problem: Authentication errors
Solutions:
  • Double-check username and password
  • Verify SSH keys are properly installed on devices
  • Check SSH key permissions: chmod 600 ~/.ssh/id_rsa
  • For Cisco devices, ensure AAA is configured correctly

No LLDP Neighbors Found

Problem: Script connects but finds no neighbors
Solutions:
  • Verify LLDP is enabled on devices
  • For Linux hosts, ensure lldpd is installed and running
  • Check if neighbors exist: run show lldp neighbors manually
  • Some devices may take time for LLDP to discover neighbors

Graphviz Errors

Problem: Static diagram generation fails
Solutions:
  • Install system Graphviz package (see installation section)
  • Verify installation: dot -V
  • Interactive HTML diagram doesn't require Graphviz

‚ö° Tips & Best Practices

  • Start Small: Test with a few devices first before scanning entire network
  • Use SSH Keys: More secure and convenient than passwords
  • Device-Specific Credentials: Use per-device authentication for mixed environments
  • Schedule Regular Scans: Run periodically to keep topology documentation updated
  • Backup Configurations: Keep a copy of your devices.json file
  • Interactive Diagrams: Use the HTML diagram for large, complex networks
  • Security: Protect credential files with proper permissions
