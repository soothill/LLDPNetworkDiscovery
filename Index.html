A comprehensive Python tool for discovering network topology using LLDP (Link Layer Discovery Protocol) and generating visual network diagrams.

## Features

- üîê Multiple authentication methods (password, SSH keys, hybrid)
- üåê Multi-vendor support (Cisco, MikroTik, HP/Aruba, Arista, Linux/Proxmox)
- üìä Multiple report formats (CSV, HTML, PNG, SVG, Interactive HTML)
- üé® Automatic network topology visualization
- üîß Device-specific credential support
- üì° Support for complex network environments

## Supported Devices

- Cisco IOS/IOS-XE/NX-OS
- MikroTik RouterOS
- HP ProCurve / Aruba OS
- Arista EOS
- Linux hosts with lldpd (Ubuntu, Debian, Proxmox, CentOS, RHEL)

## Installation

```bash

pip install netmiko graphviz
```

1. Copy `devices.json.example` to `devices.json`

```
üêõ Troubleshooting

Permission Denied (publickey)

Problem: Can't push to GitHub
Solutions:
  • Make sure you've added your SSH key to GitHub
  • Verify SSH connection: ssh -T git@github.com
  • Or use HTTPS instead: git remote set-url origin https://github.com/USERNAME/REPO.git

Repository Already Exists

Problem: Error when creating repository
Solution: Choose a different repository name or delete the existing one from GitHub

Accidentally Committed Credentials

‚ö†Ô∏è SECURITY ISSUE: If you accidentally committed credentials:
  1. Immediately change all passwords/keys that were exposed
  2. Remove the file: git rm --cached devices.json
  3. Add to .gitignore: echo "devices.json" >> .gitignore
  4. Commit: git commit -m "Remove credentials file"
  5. Push: git push
  6. Consider making the repository private

Large Files Error

Problem: Error about file size when pushing
Solution: GitHub has a 100MB file limit. Make sure large reports are in .gitignore
```
